## Getting Started

### 1. Setup Project
```bash
git clone git@github.com:alexlapinski/personal-tsdb.git
cd personal-tsdb/
npm install

cd packages/weather-scraper/
npm install
```

### 2.a. Start local dev server
```bash
npm run dev
```

### 2.b. Build & Start Local Server
```bash
npm run build
npm start
```

### 2.c. Build Docker Image & Start Docker Image
```bash
npm run docker-build
npm run docker-run
```
